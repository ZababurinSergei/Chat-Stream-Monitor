## 3. Изменение кода

**3.1 Запрет на прямое изменение**

Запрещается:

- Редактирование, модификация, адаптация или изменение исходного кода
  Программного обеспечения
- Создание производных работ на основе Программного обеспечения
- "Форкание" (forking) репозитория с последующим изменением кода

**3.2 Исключительный способ внесения изменений**

Единственный разрешённый способ внесения изменений — через **Pull Request** в
официальный репозиторий Автора:

1. Создайте форк репозитория (без права на дальнейшее изменение после создания
   PR)
2. Внесите изменения в своём форке
3. Отправьте Pull Request в основной репозиторий
4. Автор рассматривает предложенные изменения
5. В случае принятия, изменения включаются в официальную версию

Все изменения, отправленные через Pull Request, автоматически передаются под
действие данной лицензии и становятся собственностью Автора.
