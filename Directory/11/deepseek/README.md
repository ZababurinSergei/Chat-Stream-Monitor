## Как открыть консоль браузера

### Google Chrome / Edge / Opera
| Действие | Комбинация клавиш |
|----------|-------------------|
| Открыть консоль | `F12` или `Ctrl + Shift + I` (Windows/Linux) |

## Структура логов модуля

Все логи модуля `chatMonitor.js` имеют единый формат:

```
🤖 [CHAT-MONITOR] ТИП_СООБЩЕНИЯ Текст сообщения
```

### Типы сообщений:

| Тип | Префикс | Цвет | Описание |
|-----|---------|------|----------|
| Ввод пользователя | `✏️ USER INPUT` | Зеленый | Сообщение, которое вводит пользователь |
| Начало ввода | `⌨️ TYPING` | Бирюзовый | Пользователь начал печатать |
| Начало задачи | `🎬 START` | Зеленый | Отправка запроса к чату |
| Окончание задачи | `🏁 END` | Синий | Получение ответа от чата |
| Вывод чата | `💬 MESSAGE` | Серый | Текст ответа ассистента |
| Ошибка | `❌ ERROR` | Красный | Ошибка при обработке |
| Валидация | `✅ VALIDATION` | Фиолетовый | Результаты проверки ответа |
| Информация | `ℹ️ INFO` | Оранжевый | Дополнительная информация |

## Фильтрация логов

### Способ 1: Фильтр по тексту (самый простой)

1. Откройте консоль браузера (`F12`)
2. Найдите поле ввода с надписью **"Filter"** или **"Фильтр"**
3. Введите нужный текст фильтра

**Поле фильтра находится здесь:**

```
┌─────────────────────────────────────────────────────────────────┐
│ Console │ Elements │ Sources │ Network │ Performance │ →       │
├─────────────────────────────────────────────────────────────────┤
│ 🔍 Filter (regex)  [?] [Default levels ▼] [Preserve log] [ ]   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  🤖 [CHAT-MONITOR] ✏️ USER INPUT Сообщение: "Привет"            │
│  🤖 [CHAT-MONITOR] ⌨️ TYPING Пользователь начал ввод            │
│  🤖 [CHAT-MONITOR] 🎬 START НАЧАЛО ЗАДАЧИ                       │
│  🤖 [CHAT-MONITOR] ℹ️ INFO Сообщение: "Привет"                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Способ 2: Фильтр по уровню логирования

В консоли есть кнопка **"Default levels"** - нажмите её и выберите нужные уровни:

- ✅ **Info** - информационные сообщения
- ✅ **Warnings** - предупреждения
- ✅ **Errors** - ошибки
- ✅ **Debug** - отладочные сообщения

## Примеры фильтров

### 1. Показать только логи модуля

**Введите в поле фильтра:**
```
[CHAT-MONITOR]
```

**Результат:** Увидете только сообщения от нашего модуля

### 2. Показать только ошибки модуля

**Введите в поле фильтра:**
```
[CHAT-MONITOR] ERROR
```

**Результат:** Только ошибки

### 3. Показать только начало и конец задач

**Введите в поле фильтра:**
```
[CHAT-MONITOR] (START|END)
```

**Результат:** Только сообщения о начале и окончании задач

### 4. Показать только сообщения пользователя

**Введите в поле фильтра:**
```
[CHAT-MONITOR] USER INPUT
```

**Результат:** Только введенные пользователем сообщения

### 5. Показать только ответы чата

**Введите в поле фильтра:**
```
[CHAT-MONITOR] MESSAGE
```

**Результат:** Только ответы ассистента

### 6. Показать только валидацию

**Введите в поле фильтра:**
```
[CHAT-MONITOR] VALIDATION
```

**Результат:** Только результаты проверки ответов

### 7. Исключить все логи модуля

**Введите в поле фильтра:**
```
-[CHAT-MONITOR]
```

**Результат:** Логи модуля скрыты, видны только остальные

### 8. Комбинированный фильтр (Regex режим)

**Сначала включите Regex:** нажмите на иконку `.*` в поле фильтра

**Затем введите:**
```
\[CHAT-MONITOR\]\s+(START|END|MESSAGE)
```

**Результат:** Только начало, конец и ответы чата

## Сохранение фильтров

### Chrome / Edge

1. Нажмите на иконку `⚙️` (Settings) в консоли
2. Выберите **"Save console filter"**
3. Введите название фильтра (например, "Chat Monitor Only")
4. Фильтр сохранится в выпадающем списке

### Firefox

1. Нажмите на иконку `📌` (Pin) рядом с полем фильтра
2. Фильтр сохранится для текущей сессии

| `[CHAT-MONITOR] VALIDATION` | Валидацию |
| `[CHAT-MONITOR] USER INPUT` | Сообщения пользователя |
| `[CHAT-MONITOR] MESSAGE` | Ответы чата |
| `-[CHAT-MONITOR]` | ВСЕ КРОМЕ логов модуля |
| `[CHAT-MONITOR] (START\|END)` | Начало и окончание |
| `[CHAT-MONITOR] (ERROR\|VALIDATION)` | Ошибки и валидацию |

## Дополнительные настройки

### Включение/выключение логов модуля

При инициализации модуля можно настроить уровень логирования:

```javascript
window.deepSeekChatMonitor = new DeepSeekChatMonitor({
    verbose: true,           // Включить все логи
    // или
    verbose: false,          // Выключить все логи
    
    // Дополнительные настройки
    logging: {
        enabled: true,       // Включить/выключить логирование
        showDebug: false,    // Показывать debug сообщения
        showInfo: true,      // Показывать info сообщения
        showValidation: true // Показывать валидацию
    }
});
```

### Очистка консоли

- **Быстрая очистка:** `Ctrl + L`
- **Очистить историю:** ПКМ → "Clear console"
- **Сохранять логи:** Включите "Preserve log" - логи не будут очищаться при перезагрузке страницы

## Полезные советы

1. **Закрепите вкладку Console** - перетащите её в отдельное окно для удобства
2. **Используйте "Preserve log"** - чтобы логи не исчезали при навигации по сайту
3. **Сохраняйте фильтры** - для быстрого переключения между режимами
4. **Комбинируйте фильтры** - с другими ключевыми словами для точного поиска
5. **Экспортируйте логи** - ПКМ → "Save as..." для сохранения всех логов в файл

## Устранение проблем

| Проблема | Решение |
|----------|---------|
| Не вижу логов модуля | Проверьте что модуль загружен: введите `window.deepSeekChatMonitor` в консоли |
| Логи не фильтруются | Убедитесь что вы ввели фильтр без лишних пробелов |
| Regex не работает | Нажмите на иконку `.*` в поле фильтра для включения regex режима |
| Логи слишком быстрые | Включите "Preserve log" для сохранения истории |
