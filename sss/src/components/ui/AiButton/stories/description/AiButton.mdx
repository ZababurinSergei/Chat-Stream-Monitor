`AiButton` — обертка над `NButton` из `naive-ui` с единым API для двух режимов:
- `default` — обычная кнопка;
- `choice` — кнопка выбора варианта.

## Default-режим

### Пропсы режима `default`
<PropsTable table={defaultModePropsTable} />

- Используется по умолчанию (`mode: 'default'`).
- Тип кнопки задается пропсом `buttonType`.
- Если `buttonType` не передан, используется `primary`.

### Примеры использования

```html
<AiButton
    button-type="primary"
    text="Сохранить"
    @click="onClick"
/>
```
### События

- `click` -> `emit('click')`

## Choice-режим
### Пропсы режима `choice`
<PropsTable table={choiceModePropsTable} />
- В этом режиме нужно передать `option`.
- `buttonType` в режиме `choice` не используется.
- В событие `click` пробрасывается выбранный вариант.

### Примеры использования

```html
<AiButton
    mode="choice"
    option="positive"
    text="Да"
    @click="onChoice"
/>
```

```ts
const onChoice = (option: 'positive' | 'negative' | 'neutral') => {
    if (option === 'positive') {
        // ...
    }
};
```

### События

- `click` -> `emit('click', option)`
