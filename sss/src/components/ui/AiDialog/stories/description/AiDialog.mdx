`AiDialog` — модальное окно выбора действия, которое управляется через `useDialog`.

## API

Метод `show` принимает объект типа `DialogShowConfig` и возвращает `Promise<'positive' | 'negative' | 'neutral' | null>`.

### Поведение кнопок

- Если передан текст для positive кнопки (поле `positiveText`), positive кнопка отрендерится.
- Если текст для какой-либо кнопки не передан, то эта кнопка не отрендерится.
- Если не передан ни один текст кнопки, используется дефолтная кнопка: `positive` с текстом `Ок`.

### Важные ограничения

- Одновременно поддерживается **только один активный диалог**.
- Если вызвать `dialog.show(...)` во время уже открытого диалога, в консоль пишется ошибка `Диалоговое окно уже открыто`, а метод вернёт `Promise.resolve(null)`.

## Пример использования

```ts
import { useDialog } from '@/composables/AiDialog/useDialog';

const dialog = useDialog()

const handle = async () => {
    const choice = await dialog.show({
        title: 'Сохранение изменений',
        content: 'Сохранить изменения?',
        positiveText: 'Да',
        negativeText: 'Нет',
        neutralText: 'Отмена',
    });

    if (choice === 'positive') {
        // логика на 'positive'
    }
    else if (choice === 'negative') {
        // логика на 'negative'
    }
    else if (choice === 'neutral') {
        // логика на 'neutral'
    }
}
```
