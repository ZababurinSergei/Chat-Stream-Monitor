# Архитектура компонентов

## Используемые инструменты
Для демонстрации архитектуры компонентов используется **Mermaid flowchart**.

## Работа с диаграммами
Для удобной работы с файлами в `docs/architecture/`:

### WebStorm
1. Установите плагин **Mermaid**
2. Откройте любой `.mmd` файл с диаграммой

### Онлайн-редактор
1. Скопируйте код из `.mmd` файла
2. Перейдите на [Mermaid Live Editor](https://mermaid.live)
3. Вставьте код для просмотра и редактирования
