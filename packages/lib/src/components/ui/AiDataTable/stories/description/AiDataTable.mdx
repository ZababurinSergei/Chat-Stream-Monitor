## Проп `meta`
### Тип `Meta`

Объект `meta` задаёт конфигурацию таблицы: редактирование, выбор строк, ключ строки, проброс пропсов в `NDataTable`, эндпоинты API и описание колонок. Компонент принимает его в пропе `meta` (см. `TableProps`).

- **`isEditable`** — разрешить редактирование ячеек (совместно с `allowEditing` у колонок).
- **`selectionMode`** — режим колонки выбора: `multiple`, `single` или `none` (по умолчанию в проекте часто `multiple`).
- **`keyExpr`** — имя поля в строке данных, используемое как идентификатор строки (если не задано — используется `id`).
- **`config`** — дополнительные пропсы naive-ui `NDataTable`; сливаются поверх внутренних значений по умолчанию.
- **`api`** — обязательный объект с URL: **`get`** для загрузки данных, **`set`** — опционально для сохранения.
- **`columns`** — массив колонок типа `UserColumn` (подписи, поля, типы данных, компоненты, редакторы, вложенные колонки).

```ts
import type { DataTableProps } from 'naive-ui';

export interface Meta {
    isEditable?: boolean;
    selectionMode?: 'multiple' | 'single' | 'none';
    keyExpr?: string;
    config?: Partial<DataTableProps>;
    api: {
        get: string;
        set?: string;
    };
    columns: UserColumn[];
}
```

### Тип `UserColumn`

Колонка — это пересечение доменных полей (`UserOnlyColumn`) и пропсов naive-ui `DataTableBaseColumn`, кроме **`key`** и **`title`**: в таблице они выставляются из **`dataField`** (ключ колонки) и **`caption`** (заголовок).

Доменные поля:

- **`allowEditing`** — разрешить редактирование ячеек в этой колонке (вместе с `meta.isEditable`).
- **`caption`** — текст заголовка колонки (становится `title` у naive-ui).
- **`component`** — конфигурация компонента отображения значения в ячейке (см. `Component` в `@/interfaces/common.ts`).
- **`editor`** — редактор ячейки (`selector` / `popover` и параметры), см. `Editor` в `@/interfaces/common.ts`.
- **`dataField`** — имя поля в объекте строки данных / ключ колонки (становится `key` у naive-ui).
- **`displayField`** — отдельное поле для отображаемого значения, если оно отличается от `dataField`.
- **`dataType`** — тип данных (`string`, `number`, `date`, `datetime`, `boolean`) — влияет на дефолтные размеры и выравнивание колонки.
- **`children`** — вложенные колонки для иерархического заголовка.

Плюс любые подходящие поля **`DataTableBaseColumn`** (ширина, `align`, `fixed`, `sorter` и т.д.), кроме переопределения `key` и `title` напрямую.

```ts
import type { DataTableBaseColumn } from 'naive-ui';
import type { Component, DataType, Editor } from '@/interfaces/common.ts';

interface UserOnlyColumn {
    allowEditing?: boolean;
    caption: string;
    component?: Component;
    editor?: Editor;
    dataField: string;
    displayField?: string;
    dataType?: DataType;
    children?: UserColumn[];
}

export type UserColumn = UserOnlyColumn & Omit<DataTableBaseColumn, 'key' | 'title'>;
```

Источник типов: `src/components/ui/AiDataTable/types.ts`.
